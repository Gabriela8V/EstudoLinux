Comando para a navegações entre diretorios:
    Cd 
    para abrir um diretorio usamos o comando: cd e o nome do diretorio
    Ex.:cd  home -> abrindo a pasta home 

    pwd
    o comando pwd irá nos mostra onde etamos, ou seja, em qual diretorio
    você está.
    ls
    o comando ls(listar) permite exibir uma lista de arquivos em um 
    diretorio. Ele possui algumas flags(opções de comandos).
    Ex.:exibir os arquivos e pastar, inclusive os ocultos
       ls -la 


    mkdir
    o  comando cria diretorios. Por isso o seu nome é mkdir(make directory)
    Ex.:mkdir dados

    Cp
    o comando Cp(copy) é responsável por realizar uma cópia
    de arquivos ou diretorios
